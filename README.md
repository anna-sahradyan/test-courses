# Одностраничный лендинг для курса

## Общая информация

- Сайт — **одностраничный лендинг**, полностью адаптивный для всех экранов.
## Технологии

- **HTML5**
- **CSS3**
- **Bootstrap 4.6**
- **JavaScript (jQuery)**
- **Google Fonts** (PT Sans)

## Навигация и функционал

1. **Якорные ссылки и плавный скролл** — учитывается высота шапки.
2. **Клик по логотипу** — прокрутка в начало страницы.
3. **Меню адаптируется для мобильных устройств** (бургер-меню).
4. **Адаптивность**:
    - Блоки перестраиваются в одну колонку на маленьких экранах.
    - Изображения масштабируются или скрываются.
    - Весь функционал сохраняется на мобильных устройствах.

## Интерактивные элементы

1. **Выпадающие блоки** в разделе "Материалы".
2. **Аккордеон** в разделе FAQ.
3. **Переключение валют** в тарифах с пересчётом цен. Каждая карточка работает независимо.
